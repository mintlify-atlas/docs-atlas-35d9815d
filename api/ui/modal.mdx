## Overview

The `Modal` class represents a popup dialog that can collect text input from users. Modals can contain multiple text input fields and are displayed as an overlay on the Discord client. Discore re-exports discord.py's Modal class.

## Class Definition

```python
from discore.ui import Modal

class Modal(discord.ui.Modal):
    """A popup form for collecting user input."""
```

## Constructor

```python
Modal(
    *,
    title: str,
    timeout: float | None = None,
    custom_id: str | None = None
)
```

<ParamField path="title" type="str" required>
  The title of the modal displayed at the top (max 45 characters).
</ParamField>

<ParamField path="timeout" type="float | None" default="None">
  The timeout in seconds before the modal expires. `None` means no timeout.
</ParamField>

<ParamField path="custom_id" type="str | None" default="None">
  A unique identifier for the modal (max 100 characters).
</ParamField>

## Usage Examples

### Basic Modal

```python
from discore.ui import Modal, TextInput
from discord import Interaction, TextStyle

class FeedbackModal(Modal):
    def __init__(self):
        super().__init__(title="Feedback Form")
        
        # Add text inputs
        self.add_item(TextInput(
            label="What's your feedback?",
            style=TextStyle.paragraph,
            placeholder="Tell us what you think...",
            required=True,
            max_length=1000
        ))
    
    async def on_submit(self, interaction: Interaction):
        feedback = self.children[0].value
        await interaction.response.send_message(
            f"Thanks for your feedback: {feedback}",
            ephemeral=True
        )

# Show the modal in response to a button click
@discord.ui.button(label="Give Feedback")
async def feedback_button(self, interaction: Interaction, button: Button):
    await interaction.response.send_modal(FeedbackModal())
```

### Modal with Error Handling

```python
class SafeModal(Modal):
    def __init__(self):
        super().__init__(title="Submit Data")
        
        self.add_item(TextInput(
            label="Data",
            placeholder="Enter data here..."
        ))
    
    async def on_submit(self, interaction: Interaction):
        try:
            data = self.children[0].value
            # Process data (might raise exception)
            result = await process_data(data)
            await interaction.response.send_message(
                f"Success: {result}",
                ephemeral=True
            )
        except ValueError as e:
            await interaction.response.send_message(
                f"Invalid data: {e}",
                ephemeral=True
            )
    
    async def on_error(self, interaction: Interaction, error: Exception):
        await interaction.response.send_message(
            "An error occurred while processing your submission.",
            ephemeral=True
        )
        # Log the error
        print(f"Modal error: {error}")
```

### Dynamic Modal

```python
class DynamicModal(Modal):
    def __init__(self, fields: list[dict]):
        super().__init__(title="Dynamic Form")
        
        # Add fields dynamically
        for field in fields[:5]:  # Max 5 fields
            self.add_item(TextInput(
                label=field.get("label", "Field"),
                placeholder=field.get("placeholder"),
                required=field.get("required", True),
                style=field.get("style", TextStyle.short)
            ))
    
    async def on_submit(self, interaction: Interaction):
        # Collect all values
        values = [child.value for child in self.children]
        
        await interaction.response.send_message(
            f"Submitted {len(values)} fields",
            ephemeral=True
        )

# Use with dynamic configuration
fields = [
    {"label": "Name", "placeholder": "Your name"},
    {"label": "Email", "placeholder": "your@email.com"},
    {"label": "Comments", "style": TextStyle.paragraph, "required": False}
]

modal = DynamicModal(fields)
await interaction.response.send_modal(modal)
```

### Validation Example

```python
class ValidatedModal(Modal):
    email_input = TextInput(label="Email", placeholder="user@example.com")
    
    def __init__(self):
        super().__init__(title="Email Form")
    
    async def on_submit(self, interaction: Interaction):
        email = self.email_input.value
        
        # Validate email format
        if "@" not in email or "." not in email:
            await interaction.response.send_message(
                "Please enter a valid email address!",
                ephemeral=True
            )
            return
        
        await interaction.response.send_message(
            f"Email {email} registered!",
            ephemeral=True
        )
```

## Limitations

- Maximum 5 TextInput components per modal
- Modal title has a 45 character limit
- Modals must be sent in response to an interaction
- Cannot be sent in response to a message component if that component already opened a modal
- Modals are ephemeral and cannot be seen by other users
- Once submitted or closed, a modal cannot be reopened

## Best Practices

1. **Keep it simple**: Only ask for essential information
2. **Use placeholders**: Help users understand what to enter
3. **Set appropriate lengths**: Use min_length and max_length to enforce valid input
4. **Validate input**: Check submitted data in on_submit before processing
5. **Handle errors**: Always implement on_error for production modals
6. **Use paragraph style**: For longer text, use TextStyle.paragraph
7. **Mark optional fields**: Set required=False for non-essential fields
